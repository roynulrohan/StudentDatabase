Student Database API

Account Commands

POST

/api/account/register

Creates new account

- Body { username, password } : String, String

Return -> Account

POST

/api/account/login

Login to existing account

- Body { username, password } : String, String

Return -> Account

Student Commands Bearer Token RequiredAccount ID Required

POST

/api/account/{accountId}/students

Add new Student

- Body { name, email, dob} : String, String, Date

Return -> Student

GET

/api/account/{accountId}/students

Get all students on your account

Return -> List<Student>

GET

/api/account/{accountId}/students/{studentId}

Get Student by ID

Return -> Student

DELETE

/api/account/{accountId}/students/{studentId}

Delete Student by ID

Return -> N/A

PUT

/api/account/{accountId}/students/{studentId}

Replace Student by ID

- Body { name, email, dob} : String, String, Date

Return -> Student

PATCH

/api/account/{accountId}/students/{studentId}

Update Student fields by ID

- Optional Parameters name, email : String, String

Return -> Student
